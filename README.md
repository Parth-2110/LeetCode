# 🚀 LeetCode 

Welcome to my **LeetCode ** repository! This repository contains my solutions to LeetCode problems that I solve as part of my **Data Structures and Algorithms (DSA)** journey.

## 🎯 Goal

* Strengthen problem-solving skills.
* Master Data Structures and Algorithms.
* Prepare for coding interviews and placements.
* Build consistency by solving problems regularly.

## 📂 Repository Structure

Each solution is organized by LeetCode problem and includes:

* ✅ Problem statement link
* ✅ Solution code
* ✅ Clean and readable implementation
* ✅ Efficient approach

## 💻 Language Used

* Java ☕

(More languages may be added in the future.)

## 📈 Progress

I continuously update this repository as I solve more problems.

Current focus:

* Arrays
* Hashing
* Two Pointers
* Binary Search
* Sliding Window
* Linked Lists
* Stacks & Queues
* Trees
* Graphs
* Dynamic Programming

## 🧠 Learning Approach

I don't just memorize solutions. My approach is:

1. Attempt the problem on my own.
2. If stuck, understand the optimal solution thoroughly.
3. Write the logic in my notes.
4. Revisit and solve it again until I can implement it independently.

## ⭐ Why this Repository?

This repository serves as:

* A record of my DSA journey.
* A place to revise important problems.
* A resource for interview preparation.
* A way to track my consistency and improvement.
